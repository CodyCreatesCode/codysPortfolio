Javascript, HTML, & CSS
OuterLimits Entertainment is a state-of-the-art digital platform constructed using the powerful combination of Javascript, HTML, and CSS. The platform serves a dual purpose; it functions as a medium for enhancing user traffic by leveraging captivating designs and user-friendly interface, and as a vibrant hub for presenting current events in an engaging manner. The resulting combination makes OuterLimits Entertainment an immersive platform for visitors to explore, interact, and stay updated with the latest happenings.

Javascript, HTML, & CSS
The Movie Search App is a bespoke digital platform designed with the optimal combination of JWT, Express, Node, PostgreSQL, EJS, Javascript, and CSS. This platform allows users to search and explore a vast library of movies with a smooth and intuitive user interface. Upon logging in with the correct credentials, users can personalize their experience by marking movies as favorites, allowing easy access to their top picks at any given moment. The app leverages the Express.js framework running on Node.js for server-side operations, while PostgreSQL handles the data management tasks, providing a robust and reliable backend solution. The frontend, enhanced with EJS, JavaScript, and CSS, ensures dynamic, interactive, and visually pleasing user interactions. In addition, JWT (JSON Web Tokens) have been employed for secure authentication, enabling users to access and manage their accounts safely. The Movie Search App is a comprehensive tool for film enthusiasts who value personalized content and a high-quality user experience.

Javascript, HTML, & CSS
The Wildfire Tracker is an innovative web-based application built utilizing a strong quartet of technologies: Javascript, HTML, CSS, and Three.js. Designed to monitor and visualize wildfire activities across the globe in real-time, it serves as a crucial tool for both awareness and precautionary purposes. The application employs Javascript for delivering interactive and dynamic functionalities. HTML and CSS work in tandem to structure the content and design the aesthetic elements of the platform, creating a responsive and visually engaging interface. The true magic of the Wildfire Tracker, however, lies in the use of Three.js. This cross-browser JavaScript library/API is used to create and display animated 3D graphics in a web browser, providing users with an immersive and interactive visualization of the world's current wildfire events. The end result is a compelling platform that takes the user beyond mere statistics, offering an interactive global view of wildfires, and thereby promoting awareness and preparedness for such events.
